# UTraffic Pista Installer

## Inno Setup UTraffic Pista (Lane) Software offline installer script.

### Installer includes the following sections and components:

- ### Prerequisites:

1. PostgreSQL 15.3 (includes C++ Redis)
2. .Net 4.6.1.
3. Node Js 18.16.1
4. PM2 (npm package)
5. Windows ISO (required to install .Net 3.5 offline)
6. NET-Framework-3.5-Offline-Installer-v2.3
7. NI-DAQ

- ### Installation Files:

8. Pista Frontend Project
9. C# Services \
   9.1 Camarógrafo \
   9.2. Conector \
   9.3. Impresora \
   9.4. Lector Manual \
   9.5. Mantenimiento \
   9.6. Panel Mensajería \
   9.7. Pista \
   9.8. RFID \
   9.9. UTC
10. Postgres Database Schemas \
    10.1 ut_utoll_tyr_vacia.backup \
    10.2 ut_utoll_tyr_log_vacia.backup
11. Server
